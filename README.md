# client-app

app for authentication, set preferences and dashboards analytics

## Project info

**edit the project locally**

```sh
git clone https://github.com/FSD-Final-Project/client-app.git
cd client-app
npm i
npm run dev
```

## Technologies

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
