# client-app

app for authentication, set preferences and dashboards analytics of Chat Assistant

## App Preview

![](public/FSD.gif)

## Running the project locally

```sh
git clone https://github.com/FSD-Final-Project/client-app.git
cd client-app
npm i
npm run dev
```

## Technologies

This project is built with:

- Vite
- TypeScript
- React
- Tailwind CSS
- Radix UI
